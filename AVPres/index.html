Hydra is...

  • An open-source repository solution
  • Written in the Ruby language
  • Using the Rails development engine
  • With a Fedora repository for content and metadata storage
  • A Solr index for search and retrieval
  • ...and a Blackight gem to provide a basic user interface

How do I use it?

  • Build your own hydra "head" --- write code!
  • Model your video content in Fedora
  • Choose an xml metdata schema, e.g. PBcore, MODS, Dublin Core, or other
  • Develop ingestion/upload workflows
  • Create a web interface
  • Get involved with the Hydra community
